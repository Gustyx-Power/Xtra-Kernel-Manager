## Xtra Kernel Manager
Xtra Kernel Manager is an advanced kernel management application designed to optimize and control the performance of Android devices. It offers a user-friendly interface to tweak CPU, GPU settings, and more while monitoring device performance in real-time.

## Features
- **CPU Management**: Adjust maximum and minimum CPU frequencies, select governors, and more.
- **GPU Management**: Adjust maximum and minimum GPU frequencies, select power levels, and more.
- **Performance Monitoring**: Real-time information on CPU and GPU clock speeds, usage, and temperature.
- **Root Management**: Supports root access via Magisk and KernelSU.
- **Custom Profiles**: Create and manage custom profiles for different modes such as Power Saving, Balanced, and Performance.

## Installation
- Android device with root access.
- Magisk or KernelSU installed on the device.

### Steps
1. **Download the APK**: [Download Xtra Kernel Manager](httpsl://github.com/your-repo/XtraKernelManager/releases)
2. **Install the APK**
3. **Grant Root Access**: Grant root permissions to the application through Magisk or KernelSU.

## Usage
1. **Open the App**: Launch Xtra Kernel Manager on your device.
2. **Monitor Performance**: View real-time performance metrics on the home screen.
3. **CPU & GPU Settings**: Navigate to the settings menu to adjust CPU and GPU parameters.
4. **Custom Profiles**: Create and manage custom profiles according to your needs.
5. **Save Settings**: Save your changes and enjoy optimized device performance.

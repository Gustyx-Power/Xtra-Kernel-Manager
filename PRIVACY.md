# Privacy Policy - Xtra Kernel Manager

**Last updated: February 5, 2026**

## Overview
Xtra Kernel Manager (XKM) is committed to protecting your privacy. This policy explains how we handle data in our application.

## Data Collection
XKM collects minimal data necessary for functionality:

### Local Data Only
- **System Information**: CPU, GPU, memory, and thermal data for optimization
- **App Usage Statistics**: For per-app profile management (stays on device)
- **Performance Metrics**: Battery and performance statistics (local storage only)
- **User Preferences**: Settings and configurations (local storage only)

### No Remote Data Collection
- **No Personal Information**: We don't collect names, emails, or personal data
- **No Analytics**: No usage analytics sent to external servers
- **No Advertising**: No ad tracking or behavioral profiling
- **No Cloud Sync**: All data remains on your device

## Permissions Usage
Required permissions are used solely for stated functionality:

- **Accessibility Service**: Game detection for overlay features only
- **System Alert Window**: Game overlay display
- **Usage Stats**: Per-app performance profiling
- **Root Access**: Kernel parameter modification (optional)

## Data Security
- All data stored locally using Android's secure storage
- No network transmission of personal data
- Open source code available for audit

## Third-Party Services
- **Firebase**: Used only for app configuration (no personal data)
- **No Analytics Services**: No Google Analytics, Facebook, or similar

## Your Rights
- All data is under your control
- Uninstalling the app removes all data
- No account creation required

## Contact
For privacy concerns: support@xms.id

## Changes
Updates to this policy will be posted in app updates and on GitHub.

Changelogs For Xtra Kernel Manager Version 1.4

Major Changes:
- Enhanced UI/UX with improved visual design and user experience: Complete redesign of the interface with modern Material Design principles, improved navigation flow, redesigned control panels, better accessibility features, and enhanced visual feedback for user interactions
- Added comprehensive CPU frequency monitoring and control system
- Implemented real-time CPU temperature monitoring
- Added intelligent battery optimization features with custom power profiles
- Implemented system-wide performance monitoring dashboard
- Enhanced memory management with automatic cleanup and optimization
- Added support for custom kernel parameter tuning

Minor Changes:
- Improved stability and error handling across all modules
- Enhanced root permission management and verification system
- Optimized app startup time and reduced memory footprint
- Enhanced security with improved permission handling



Changelogs For Xtra Kernel Manager Version 1.3
- Resolved an issue where Tuning functionalities were not operational in Version 1.2.
- Implemented CPU Core Control, allowing users to manage individual CPU core states.
- Introduced Ram Control, providing options to optimize RAM usage.
- Removed the Terminal feature.
- Added a Miscellaneous section (currently under development) for future enhancements.
- Incorporated an "Adjust Swap Size" feature (to be released soon), enabling users to modify the ZRAM swap size.
- Update and Tidy up the UI: Refined the user interface for a more intuitive and visually appealing experience. This includes improved layout, clearer navigation, and updated icons to enhance usability.
